SPRITE VARIANT BACKGROUND PACK

This pack contains eight coordinated, text-free backgrounds:
Base, Gold, Gummy, Galaxy, Cube, Gem, Quack, and Holofoil.

RECOMMENDED PLACEMENT

Use the files in well-1200x1200 behind the sprite artwork only.
This keeps sprite labels, collection controls, and mastery controls readable
and gives the row the same clean panel effect as the supplied reference.

ALTERNATE PLACEMENT

Use the files in card-1200x1500 as the background for the entire sprite card.
This produces a more dramatic result, but the text and controls may need a
dark translucent surface for consistent contrast.

IMAGE SETTINGS

- Fit: Cover
- Position: Center
- Do not stretch
- WebP is intentional: it stays sharp while loading much faster than PNG.
- Each variant should reuse one shared file across every sprite group.
  Re-uploading the same image into dozens of individual cards can waste
  browser storage and make the public design unnecessarily large.
